# NitroEdge Garage Landing Page

A dark, aggressive, premium car detailing and performance auto care landing page concept created as a Fiverr portfolio project.

## Brand concept

NitroEdge Garage is a performance-focused auto care brand offering premium detailing, ceramic coating, and vehicle enhancement services for drivers who want their cars to look sharper, cleaner, and more powerful.

## Features

- Dark metallic hero section
- Red accent visual system
- Sticky responsive navigation
- Mobile hamburger menu
- Service cards with hover effects
- Before/after results section
- Mobile package carousel with active dots
- Testimonials carousel on mobile
- Booking process section
- FAQ accordion
- Booking form validation
- Toast notification feedback
- Scroll reveal animations
- Open Graph social preview
- Sitemap and robots setup
- Custom loading screen
- Custom 404 page

## Tech stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide React
- Vercel-ready deployment

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.
